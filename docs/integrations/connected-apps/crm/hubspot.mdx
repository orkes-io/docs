# HubSpot Integration with Orkes Conductor

:::info Available since
- v5.3.0 and later
:::

Integrating HubSpot with Orkes Conductor lets you build workflows that interact with your HubSpot CRM using the following operations:

- Create Contact
- Get Contact
- List Contacts
- Search Contacts
- Search Contacts by Email
- Update Contact
- Delete Contact
- Create Company
- Get Company
- List Companies
- Search Companies
- Search Companies by Domain
- Update Company
- Delete Company
- Get Deal Pipelines
- Get Ticket Pipelines
- Create Deal
- Get Deal
- List Deals
- Search Deals
- Update Deal
- Delete Deal
- Create Ticket
- Get Ticket
- List Tickets
- Update Ticket
- Delete Ticket
- Create Note
- Get Note
- List Notes
- Delete Note
- Create Association
- Get Associations
- Get Owners
- Get Form Fields
- Submit Form

This guide explains how to set up and use the integration. Here’s an overview:

1. Get the required credentials from HubSpot.
2. Configure a new HubSpot integration in Orkes Conductor.
3. Use HubSpot integration in workflows.

## Step 1: Get the HubSpot credentials

To integrate HubSpot with Orkes Conductor, create a private (legacy) application in Hubspot and retrieve the access token.

**To retrieve the access token:**

<ol>
<li>Log in to <a href="https://app.hubspot.com/">HubSpot</a>.</li>
<li>Select the settings icon (⚙) in the top navigation bar.</li>
<li>Go to <strong>Integrations</strong> &gt; <strong>Legacy Apps</strong>.</li>
<li>Select <strong>Create legacy app</strong>.</li>
<li>Enter a name and description for your app.</li>
<li>Go to the <strong>Scopes</strong> tab and add the required scopes based on your use case.</li>
</ol>

| Scope | Operations |
| ----- | ---------- |
| crm.objects.contacts.read | Get Contact, List Contacts, Search Contacts, Search Contacts by Email | 
| crm.objects.contacts.write | Create Contact, Update Contact, Delete Contact | 
| crm.objects.companies.read | Get Company, List Companies, Search Companies, Search Companies by Domain | 
| crm.objects.companies.write | Create Company, Update Company, Delete Company | 
| crm.objects.deals.read | Get Deal, List Deals, Search Deals, Get Deal Pipelines | 
| crm.objects.deals.write | Create Deal, Update Deal, Delete Deal | 
| crm.objects.owners.read | Get Owners | 
| tickets | Create Ticket, Get Ticket, List Tickets, Update Ticket, Delete Ticket, Get Ticket Pipelines | 
| forms | Get Form Fields, Submit Form | 

<ol start="7">
<li>Select <strong>Create app</strong>.</li>
<li>Go to the <strong>Auth</strong> tab and copy the <strong>Access Token</strong>.</li>
</ol>

## Step 2: Add an integration for HubSpot

After obtaining the credentials, add a HubSpot integration to your Conductor cluster.

<ol>
<li>Go to <strong>Integrations</strong> &gt; <strong>Connected Apps</strong> from the left navigation menu on your Conductor cluster.</li>
<li>Select <strong>+ New integration</strong>.</li>
<li>In the <strong>CRM</strong> section, choose <strong>HubSpot</strong>.</li>
<li>Select <strong>+ Add</strong> and enter the following parameters:</li>
</ol>

| Parameter | Description | Required/Optional |
| --------- | ----------- | ----------------- |
| Integration name | A name for the integration. | Required. | 
| API Key | The Access Token copied in [Step 1](/content/integrations/hubspot#step-1-get-the-hubspot-credentials). | Required. | 
| Description | A description of the integration. | Required. | 

<ol start="5">
<li>(Optional) Toggle the <strong>Active</strong> button off if you don’t want to activate the integration instantly.</li>
<li>Select <strong>Save</strong>.</li>
</ol>

This creates the integration successfully.

## Step 3: Use HubSpot integration in workflows

Once the integration is ready, this can be used directly within the workflows. 

**To use HubSpot integration in a workflow:**

<ol>
<li>Go to <strong>Definitions</strong> &gt; <strong>Workflow</strong> from the left navigation menu on your Conductor cluster.</li>
<li>Select <strong>+ Define workflow</strong>.</li>
<li>In your workflow, select the (<strong>+</strong>) icon and select <strong>Connected Apps</strong>.</li>
<li>In <strong>Add Task</strong> panel on the right, search for <strong>HubSpot</strong>, and select the integration created in <a href="/content/integrations/hubspot#step-2-add-an-integration-for-hubspot">Step 2</a>.</li>
</ol>
