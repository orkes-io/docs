# Task Reference
Reference documentation for all tasks, including behavior, configuration options, and examples in Orkes Conductor.

Use the task reference when you need exact behavior for worker tasks, operators, system tasks, and AI task types.

Reference pages explain what each task does, how to configure it, what it returns, and which related primitives to use with it.
