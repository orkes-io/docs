# AI / LLM Integrations
Learn how to connect your Conductor cluster to AI and LLM providers and configure models for use in AI-powered workflows.

Use these integrations when workflows need to call LLM providers for chat, completion, extraction, classification, or agents.

Configure model providers such as OpenAI, Anthropic, Bedrock, Vertex AI, Gemini, Mistral, Cohere, Grok, Perplexity, Hugging Face, or Ollama.

## Recommended path

If you are unsure where to begin, start with **Ollama**, **Azure Open AI**, **Open AI**, **Perplexity**, **Grok**. These pages cover the most common entry points for this section.

## When to use this section

Use these pages to answer implementation questions, compare available primitives, and find the exact guide or reference page for the next step in your Orkes Conductor project.
