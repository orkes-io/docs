# Integration Catalog
Learn how to connect Conductor with AI providers, vector databases, message brokers, cloud platforms, databases, and external services.

Use this catalog when you need Conductor to connect workflows to models, queues, databases, cloud services, email, or Git.

Choose an integration by system type, configure credentials once, then reuse it from workflows, AI tasks, system tasks, and remote services.

## Recommended path

If you are unsure where to begin, start with **AI / LLM**, **Vector Databases**, **Cloud Providers**, **RDBMS**, **Email/Git**. These pages cover the most common entry points for this section.

## When to use this section

Use these pages to answer implementation questions, compare available primitives, and find the exact guide or reference page for the next step in your Orkes Conductor project.
