# Alerting Tasks
Alerting tasks send notifications from workflows through external services such as messaging or incident management platforms in Orkes Conductor.

Use alerting tasks when workflow failures, checks, or operational events should notify incident-management systems.

Combine alerting tasks with retries, timeouts, rate limits, and compensation workflows for production operations.
